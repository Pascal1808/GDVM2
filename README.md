# GDVM2


# space peggle

# beschrijving
de speler schiet een meteor af op verschillende planeten per level andere planeten er bij elke planeet heeft meer hitpoints

# gameplay kern
* bal: een meteor
* targets en bumps: planeten die elk level anders zijn
* score: in planetcoins
* doel: het eind level verslaan

# stijl en sfeer
kleurrijk, cartoony met geluiden van dingen die kapot gaan

# level structuur
bovenaan: schietplek van de meteoren
midden: het speelveld
onderaan: de shop om je meteor te upgraden
